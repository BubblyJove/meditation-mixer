# Privacy Policy

**Last updated: January 2024**

## Our Commitment

Meditation Mixer is designed with privacy as a core principle. We believe your meditation practice is personal, and your data should stay that way.

## Data Collection

**We collect nothing.**

- No personal information
- No usage analytics
- No crash reporting (unless you explicitly enable it)
- No advertising identifiers
- No location data
- No device fingerprinting

## Data Storage

All data stays on your device:

- **Presets**: Stored locally in app database
- **Session history**: Stored locally for streak tracking
- **Settings**: Stored locally in app preferences
- **Imported audio**: Referenced from your device storage (not copied)

## Network Access

This app requires **no internet connection** to function.

- No data is transmitted to any server
- No cloud sync
- No remote services
- No external API calls

## Permissions

The app requests only essential permissions:

| Permission | Purpose |
|------------|---------|
| `FOREGROUND_SERVICE` | Keep audio playing when screen is off |
| `WAKE_LOCK` | Prevent device sleep during playback |
| `POST_NOTIFICATIONS` | Show playback controls (Android 13+) |

We do **not** request:

- Storage access (we use the Storage Access Framework for user-initiated file imports)
- Internet access
- Location
- Camera
- Microphone
- Contacts
- Phone state

## Third-Party Services

There are no third-party services, SDKs, or libraries that collect user data.

## Audio Files

When you import audio files:

- Files remain in their original location
- We store only a reference (URI) to the file
- Files are never uploaded or shared
- You can revoke access at any time through Android settings

## Data Deletion

To delete all app data:

1. Go to Android Settings
2. Apps → Meditation Mixer
3. Storage → Clear Data

Or simply uninstall the app.

## Children's Privacy

This app does not knowingly collect data from anyone, including children under 13.

## Changes to This Policy

If we ever change this policy, we will:

1. Update the "Last updated" date
2. Include changes in the app changelog
3. Notify users via in-app notice (if the app has been updated to include such notices)

## Open Source

This app is open source. You can verify our privacy claims by reviewing the code yourself.

## Contact

For privacy questions or concerns, please open an issue on our GitHub repository.

---

**Summary**: We don't collect, store, or transmit any of your data. Everything stays on your device. That's it.
